# System Design Questions

### How would you approach power management in a battery-constrained embedded system, particularly one that needs to run for weeks or months.

### Imagine you have to implement a watchdog mechanism in an embedded system to ensure the system recovers from a fault. How would you design and implement that effectively?

### Imagine you need to architect a system with a central hub/control center that monitors and controls multiple devices on the its network simultaneously. How would you design the data flow and architecture so that it scales with more devices in the future?

### Imagine you have to design a fault-tolerant communication protocol for a satellite that occasionally experiences intermittent connectivity. How would you structure the protocol to ensure reliable delivery despite those gaps?

### How would you debug and optimize an embedded system that's experiencing unexpected latency spikes in its real-time response?

### Explain how you would implement a UART-based communication protocol in an embedded system, and explain what considerations you'd have to keep in mind for timing and reliability.

### Draw a state-machine for a opening and closing a garage door. How would you modify it for safety, to prevent the door closing on someone?
